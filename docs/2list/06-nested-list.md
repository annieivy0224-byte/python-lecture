# 2-6 巢狀清單與雙層迴圈

## 什麼是巢狀清單？

巢狀清單（nested list）指的是「清單中的元素本身也是清單」，常用來表示表格、矩陣或多筆資料。

```python
students = [
    ['小明', 85, 90],
    ['小華', 78, 82],
    ['小美', 92, 88]
]
```

上面的例子中，每一筆資料代表一位學生：`[姓名, 國文成績, 英文成績]`。

## 存取巢狀清單的元素

要取出巢狀清單中的元素，需要使用**兩層索引**：

```python
students = [
    ['小明', 85, 90],
    ['小華', 78, 82],
    ['小美', 92, 88]
]

print(students[0])        # ['小明', 85, 90]
print(students[0][0])     # 小明
print(students[0][1])     # 85
print(students[2][2])     # 88
```

`students[0][1]` 可以理解為：「先取 `students` 的第 0 個元素（`['小明', 85, 90]`），再取其第 1 個元素（`85`）」。

## 使用單層 for 迴圈逐筆取出

```python
students = [
    ['小明', 85, 90],
    ['小華', 78, 82],
    ['小美', 92, 88]
]

for s in students:
    print(s)
```

執行結果：

```
['小明', 85, 90]
['小華', 78, 82]
['小美', 92, 88]
```

## 使用雙層 for 迴圈

若想要進一步處理每筆資料中的每個元素，可以使用雙層迴圈：

```python
students = [
    ['小明', 85, 90],
    ['小華', 78, 82],
    ['小美', 92, 88]
]

for s in students:
    for score in s[1:]:
        print(score, end=' ')
    print()
```

執行結果：

```
85 90
78 82
92 88
```

## 範例：計算每位學生的平均成績

```python
students = [
    ['小明', 85, 90],
    ['小華', 78, 82],
    ['小美', 92, 88]
]

for s in students:
    name = s[0]
    scores = s[1:]
    avg = sum(scores) / len(scores)
    print(f'{name} 的平均成績為 {avg:.1f}')
```

執行結果：

```
小明 的平均成績為 87.5
小華 的平均成績為 80.0
小美 的平均成績為 90.0
```

## 範例：建立九九乘法表（雙層迴圈 + 巢狀清單）

```python
table = []

for i in range(1, 10):
    row = []
    for j in range(1, 10):
        row.append(i * j)
    table.append(row)

print(table)
```

執行結果（節錄）：

