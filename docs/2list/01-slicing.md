# 2-1 切片 (Slicing) 與 -1 索引

清單（list）是 Python 中最常用的資料結構之一，可以儲存一系列的資料。本節將複習並進階介紹清單的索引（index）與切片（slicing）操作。

## 基本索引

清單的每個元素都有對應的索引值，從 `0` 開始計算。

```python
fruits = ['apple', 'banana', 'cherry', 'durian', 'mango']

print(fruits[0])   # apple
print(fruits[2])   # cherry
```

## -1 指標值

除了從 `0` 開始的正向索引，Python 也支援**負索引**，`-1` 代表清單的最後一個元素，`-2` 代表倒數第二個，以此類推。

```python
fruits = ['apple', 'banana', 'cherry', 'durian', 'mango']

print(fruits[-1])   # mango
print(fruits[-2])   # durian
```

::: tip 小技巧
當不確定清單長度，但想取得最後一個元素時，使用 `fruits[-1]` 比 `fruits[len(fruits)-1]` 更簡潔。
:::

## 切片 (Slicing)

切片可以取出清單中的一段連續內容，語法為：

```python
list[start:end:step]
```

- `start`：起始索引（包含），預設為 `0`
- `end`：結束索引（**不包含**）
- `step`：間隔步數，預設為 `1`

```python
nums = [10, 20, 30, 40, 50, 60, 70]

print(nums[1:4])     # [20, 30, 40]
print(nums[:3])      # [10, 20, 30]
print(nums[3:])      # [40, 50, 60, 70]
print(nums[:])       # [10, 20, 30, 40, 50, 60, 70]（複製整個清單）
```

::: warning 常見錯誤：end 是「不包含」
`nums[1:4]` 取的是索引 1、2、3 的元素，**不包含**索引 4。
初學者常誤以為會取到索引 4，請特別注意！
:::

## 使用 step 進行切片

```python
nums = [10, 20, 30, 40, 50, 60, 70]

print(nums[::2])     # [10, 30, 50, 70]，每隔一個取一次
print(nums[1::2])    # [20, 40, 60]
print(nums[::-1])    # [70, 60, 50, 40, 30, 20, 10]，反轉整個清單
```

## 負索引切片

```python
nums = [10, 20, 30, 40, 50, 60, 70]

print(nums[-3:])     # [50, 60, 70]，取最後三個元素
print(nums[:-2])     # [10, 20, 30, 40, 50]，去掉最後兩個元素
print(nums[-4:-1])   # [40, 50, 60]
```

## 範例：取出清單頭尾元素

```python
data = [3, 7, 1, 9, 4, 6]

first = data[0]
last = data[-1]

print(f'第一個元素：{first}')
print(f'最後一個元素：{last}')
```

執行結果：

```
第一個元素：3
最後一個元素：6
```

## 本節方法一覽

| 語法                | 說明                     |
| ------------------- | ------------------------ |
| `list[i]`         | 取索引 i 的元素          |
| `list[-1]`        | 取最後一個元素           |
| `list[start:end]` | 取 start 到 end-1 的元素 |
| `list[::step]`    | 依間隔步數取元素         |
| `list[::-1]`      | 反轉清單（產生新清單）   |
