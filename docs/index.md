# 🌰 的 Python 初階課程講義

本講義涵蓋資料型態、控制流程、List 與 String 進階操作、進階應用(tuple/set/dict)，以及實戰題庫練習，適合初學者循序漸進學習。講義有任何不完善的地方可以告知我~~

## 課前準備

本課程使用 Python 語言，建議使用 Visual Studio Code 搭配 Python 延伸套件進行練習。

::: tip 學習建議

1. 沒有程式基礎的同學，請先從「基礎內容」開始，依序學習資料型態與控制流程。
2. 每個章節都包含「範例程式」與「自主練習」，請務必動手實作。
3. 題庫練習章節提供完整題目與解答，建議先自行嘗試後再對照解答。
4. 若對某個方法不熟悉，歡迎提問，也可以使用 `help()` 或查閱 Python 官方文件或使用 AI。
   :::

## 課程大綱

- **零、基礎內容**：整數/浮點數、布林與邏輯運算、字串基礎、if 條件式、while/for 迴圈
  - 捷徑：[1-1 整數與浮點數](/1basic/01-numbers)
- **一、List 進階操作**：切片、排序、新增刪除、成員判斷、List Comprehension、巢狀清單
  - 捷徑：[2-1 切片 (Slicing) 與 -1 索引](/2list/01-slicing)
- **二、String 進階操作**：切片、格式化、分割合併、去除空白、查找與計數、字串與清單轉換
  - 捷徑：[3-1 字串切片與格式化輸出](/3string/01-slicing-format)
- **三、進階應用：其他資料型態**：tuple（元組）、set（集合）、dict（字典）
  - 捷徑：[4-1 元組 tuple](/4advanced/01-tuple)
- **四、實戰題庫練習**：字串題、清單題、綜合題與詳解
  - 捷徑：[5-1 字串基礎練習題](/5practice/01-string-basics)

## 適合對象

- **完全初學者**：請從「基礎內容」開始，循序漸進學習。
- **已有基礎、想加強 List/String 操作的同學**：可直接從「List 進階操作」或「String 進階操作」開始。
- **想了解更多資料型態的同學**：可參考「進階應用」章節中的 tuple、set、dict。
