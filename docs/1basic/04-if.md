# 1-4 if 條件式

`if` 條件式可以依照不同的條件，執行不同的程式區塊，是程式設計中最基本的「決策」工具。

## 只有一個 if

```python
score = 75

if score >= 60:
    print('及格')
```

當 `score >= 60` 為 `True` 時，才會執行縮排內的程式碼；若為 `False`，則什麼都不會發生。

::: warning Python 的縮排規則
Python 使用**縮排（indentation）**來表示程式區塊，通常使用 4 個空格。縮排錯誤會導致 `IndentationError`。
:::

## if 與 else

```python
score = 45

if score >= 60:
    print('及格')
else:
    print('不及格')
```

## if...elif...else

當有多個條件需要依序判斷時，使用 `elif`（else if 的縮寫）。

```python
score = 85

if score >= 90:
    grade = 'A'
elif score >= 80:
    grade = 'B'
elif score >= 70:
    grade = 'C'
elif score >= 60:
    grade = 'D'
else:
    grade = 'F'

print(f'成績等級：{grade}')
```

執行結果：

```
成績等級：B
```

::: tip elif 的判斷順序
`elif` 會**依序**判斷，一旦符合某個條件就不會再往下檢查。因此條件的順序很重要：上面的例子若把 `score >= 60` 放在最前面，那麼即使 `score` 是 `100`，也會被判斷為 `'D'`。
:::

## if 內又有 if（巢狀 if）

```python
age = 20
has_ticket = True

if age >= 18:
    if has_ticket:
        print('可以入場')
    else:
        print('請先購票')
else:
    print('未滿18歲，禁止入場')
```

## not、and、or 與 if 的組合

```python
username = 'admin'
password = '1234'

if username == 'admin' and password == '1234':
    print('登入成功')
else:
    print('帳號或密碼錯誤')
```

```python
weather = 'rain'

if not weather == 'sunny':
    print('記得帶雨傘')
```

## 範例：判斷三角形的類型

```python
a, b, c = 3, 3, 3

if a == b == c:
    print('等邊三角形')
elif a == b or b == c or a == c:
    print('等腰三角形')
else:
    print('一般三角形')
```
