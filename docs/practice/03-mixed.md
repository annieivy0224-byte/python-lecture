# 3-3 字串與清單綜合練習題

本節練習結合字串與清單的操作，是常見的綜合應用題型。

## 第 1 題：將句子拆解成單字清單並排序

請輸入一句英文句子，將其拆解成單字清單，並依照字母順序（忽略大小寫）排序後輸出。

```
輸入範例：the Quick brown Fox jumps
輸出範例：['brown', 'Fox', 'jumps', 'Quick', 'the']
```

## 第 2 題：統計字元出現頻率

請輸入一個字串，統計每個字元（不分大小寫，忽略空白）出現的次數，並依照出現次數由高到低排序輸出前 3 個字元。

```
輸入範例：hello world
輸出範例：
l: 3 次
o: 2 次
h: 1 次
```

## 第 3 題：將清單資料轉換為一行字串並輸出

給定一個學生成績清單，每筆資料為 `[姓名, 成績]`，請將其轉換成 `'姓名:成績'` 的格式，並用逗號連接成一行字串。

```python
data = [['Amy', 90], ['Bob', 85], ['Cathy', 95]]
```

```
輸出範例：Amy:90,Bob:85,Cathy:95
```

## 第 4 題：解析 CSV 格式字串並計算平均

給定一行 CSV 格式的字串，第一個欄位為姓名，其餘欄位為成績，請計算並輸出該學生的平均成績。

```
輸入範例：王小明,85,90,78,92
輸出範例：王小明 的平均成績為 86.25
```

## 第 5 題：找出句子中出現次數最多的單字

請輸入一句英文句子（單字以空白分隔），找出其中出現次數最多的單字（若有多個則輸出第一個出現的）。

```
輸入範例：the cat sat on the mat the cat ran
輸出範例：出現次數最多的單字是 "the"，出現了 3 次
```

## 第 6 題：將多行資料轉換為巢狀清單並篩選

給定多行 CSV 格式字串資料，每行格式為 `姓名,國文,英文,數學`，請將其轉換為巢狀清單，並篩選出三科平均成績達 `80` 分以上的學生姓名。

```python
lines = [
    '王小明,85,90,78',
    '李小華,60,55,70',
    '陳小美,92,88,95'
]
```

```
輸出範例：平均達80分的學生：['陳小美']
```

## 第 7 題：身分證字號簡易檢查

請輸入一個字串，依序檢查：
1. 字串長度是否為 10
2. 第一個字元是否為英文字母
3. 第 2 個字元之後是否全為數字

若三項條件皆符合，輸出「格式正確」，否則輸出「格式錯誤」。

```
輸入範例：A123456789
輸出範例：格式正確

輸入範例：A12B456789
輸出範例：格式錯誤
```

## 第 8 題：將數字清單轉換為逗號分隔字串並反轉

給定一個數字清單，將其轉換為以逗號分隔的字串，再將該字串反轉後輸出。

```
輸入範例：[1, 2, 3, 4, 5]
輸出範例：原字串：1,2,3,4,5
        反轉後：5,4,3,2,1
```

::: tip 提示
注意第 8 題反轉後的結果是針對「字串」整體反轉，而不是將清單中每個數字的順序反轉後再合併（兩者結果不同，請仔細觀察輸出範例）。
:::
