# 3-2 清單基礎練習題

本節提供與清單操作相關的基礎練習題，建議先自行思考並動手撰寫程式，再參考 [解答與說明](./04-answers)。

## 第 1 題：找出最大值與最小值

給定一個數字清單，請找出其中的最大值與最小值（**不可**使用內建函式 `max()` 與 `min()`，請使用迴圈逐一比較）。

```
輸入範例：[12, 45, 7, 89, 23, 56]
輸出範例：最大值：89，最小值：7
```

## 第 2 題：計算清單總和與平均

給定一個數字清單，請計算其總和與平均值（平均取小數點後兩位）。

```
輸入範例：[80, 92, 75, 88, 60]
輸出範例：總和：395，平均：79.00
```

## 第 3 題：移除清單中的重複元素

給定一個包含重複數字的清單，請移除重複的元素，並保留原本的出現順序。

```
輸入範例：[1, 3, 2, 3, 1, 4, 2, 5]
輸出範例：[1, 3, 2, 4, 5]
```

## 第 4 題：篩選及格名單

給定一個成績清單，請篩選出所有大於或等於 `60` 分的成績，組成新的清單。

```
輸入範例：[55, 78, 60, 45, 90, 59, 100]
輸出範例：及格名單：[78, 60, 90, 100]
```

## 第 5 題：清單反轉（不使用 reverse() 或切片）

給定一個清單，請使用迴圈將其反轉（不可使用 `reverse()` 或 `[::-1]`）。

```
輸入範例：[1, 2, 3, 4, 5]
輸出範例：[5, 4, 3, 2, 1]
```

## 第 6 題：兩個清單合併並排序

給定兩個數字清單，請將它們合併成一個清單，並由小到大排序。

```
輸入範例：[3, 1, 4]、[2, 6, 5]
輸出範例：[1, 2, 3, 4, 5, 6]
```

## 第 7 題：計算每個數字出現的次數

給定一個數字清單，請統計每個不同數字出現的次數，並依數字大小由小到大印出。

```
輸入範例：[1, 2, 2, 3, 1, 2, 4]
輸出範例：
1: 2 次
2: 3 次
3: 1 次
4: 1 次
```

## 第 8 題：將清單中所有數字加總並判斷奇偶

給定一個數字清單，計算所有元素的總和，並判斷該總和是奇數還是偶數。

```
輸入範例：[1, 2, 3, 4, 5]
輸出範例：總和為 15，是奇數
```

## 第 9 題：清單中的偶數與奇數分類

給定一個數字清單，請使用 list comprehension 將其分為偶數清單與奇數清單。

```
輸入範例：[1, 2, 3, 4, 5, 6, 7, 8]
輸出範例：
偶數：[2, 4, 6, 8]
奇數：[1, 3, 5, 7]
```

## 第 10 題：二維成績表計算每列總分

給定一個巢狀清單，每筆資料為 `[姓名, 國文, 英文, 數學]`，請計算並印出每位學生的總分。

```python
students = [
    ['Amy', 80, 90, 85],
    ['Bob', 70, 65, 88],
    ['Cathy', 95, 92, 99]
]
```

```
輸出範例：
Amy 的總分：255
Bob 的總分：223
Cathy 的總分：286
```
