# 3-1 字串基礎練習題

本節提供與字串操作相關的基礎練習題，建議先自行思考並動手撰寫程式，再參考 [解答與說明](./04-answers)。

## 第 1 題：反轉字串

請輸入一個字串，將其反轉後輸出。

```
輸入範例：Python
輸出範例：nohtyP
```

## 第 2 題：判斷回文

請輸入一個英文字串，判斷它是否為回文（順著讀與反過來讀都相同）。

```
輸入範例：level
輸出範例：level 是回文

輸入範例：python
輸出範例：python 不是回文
```

## 第 3 題：計算字串中母音的數量

請輸入一個英文字串，計算其中母音字母（`a`, `e`, `i`, `o`, `u`，不分大小寫）出現的總次數。

```
輸入範例：Hello World
輸出範例：母音數量：3
```

## 第 4 題：將姓名格式化

請輸入英文全名（格式為 `名 姓`，例如 `john smith`），將其轉換為「姓, 名」格式，且首字母大寫。

```
輸入範例：john smith
輸出範例：Smith, John
```

## 第 5 題：統計單字數量

請輸入一句英文句子，計算其中總共有幾個單字（單字以空白分隔）。

```
輸入範例：I love Python programming
輸出範例：單字數量：4
```

## 第 6 題：移除字串中的數字

請輸入一個包含字母與數字的字串，移除其中所有數字字元，只保留字母與其他字元。

```
輸入範例：a1b2c3d4
輸出範例：abcd
```

## 第 7 題：將字串中的空白替換為下底線

請輸入一句英文句子，將其中所有空白字元替換為 `_`。

```
輸入範例：Hello Python World
輸出範例：Hello_Python_World
```

## 第 8 題：身分證字號開頭字母轉縣市（簡化版）

給定字串 `'A123456789'`（第一個字元為英文字母），請判斷第一個字元是否為英文字母，且字串總長度是否為 10。若符合則印出「格式正確」，否則印出「格式錯誤」。

```
輸入範例：A123456789
輸出範例：格式正確

輸入範例：12345
輸出範例：格式錯誤
```

## 第 9 題：將日期字串轉換格式

請輸入格式為 `YYYY-MM-DD` 的日期字串（例如 `2024-06-13`），將其轉換為 `YYYY年MM月DD日` 的格式輸出。

```
輸入範例：2024-06-13
輸出範例：2024年06月13日
```

## 第 10 題：找出字串中最長的單字

請輸入一句英文句子，找出其中最長的單字並輸出（若有多個相同長度的最長單字，輸出第一個出現的）。

```
輸入範例：I love programming in Python
輸出範例：最長的單字是：programming
```
